# Containerized Multi-Service App

A full-stack Task Manager built with React, Node.js & PostgreSQL.

## Tech Stack
- Frontend: React + Nginx
- Backend: Node.js + Express
- Database: PostgreSQL
- DevOps: Docker, Kubernetes, GitHub Actions

## Progress
### Week 1 ✅
- [x] Node.js Express backend created
- [x] REST API routes: GET, POST, PATCH, DELETE

### Week 2 ✅

- [x] Nginx proxy config added

## Run locally
Terminal 1: cd backend && npm run dev
Terminal 2: cd frontend && npm start
Visit: http://localhost:3000
